A Food Blogging theme called Astrif.

== Description ==

Hi. I'm a food blogging theme called Astrif. I'm a theme for personal blogging which suitable for people who love sharing her all experience about food. I can deliver your interesting experience stories to the world. To let everyone know how wonderful your are.
I will helps present your content in the most compelling way, so it can make people fall in love.
That's what I'm here for.

Features :
* Well-commented, modern, HTML5 templates.
* Build with Bootstrap
* Build with LESS
* 6+ coolest widget
* Skin available
* Clean and elegant

== Installation ==

It's just html+css theme

== Frequently Asked Questions ==

Astrif uses less - css preprocessor
